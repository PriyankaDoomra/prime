Prime Number

What is a Prime Number?

A prime number is a whole number whose only factors are 1 and the number itself. A factor is a whole number that can be divided evenly into another number. Numbers that have more than two factors are called composite numbers. The number 1 is neither prime nor composite.

How to Find Prime Numbers?

If you would like to know whether any given number is prime, follow these steps:

> Write all the factors of the given number.

> Count the factors.

> If the number of factors is EXACTLY 2 then the given number is prime.
